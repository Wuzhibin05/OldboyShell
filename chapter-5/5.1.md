​																	**表5-2 Shell中常见的算术运算命令**

| 运算操作符与运算命令 | 意义                                                 | 记忆方法   |
| -------------------- | ---------------------------------------------------- | ---------- |
| (())                 | 用于**整数运算**的常用运算符，效率很高               | 整数运算   |

| expr                 | 可以用于整数运算，但很还有很多其他的额外功能         |            |
| bc                   | linux下的一个计算器程序（适合整数和小数运算）        | 计算器     |
| $[]                  | 用于整数运算                                         |            |
| awk                  | awk既可以用于整数运算，也可以用于小数运算            | 整数和小数 |
| declare              | 定义变量值和属性，-i参数可以用户定义整形变量，做运算 |            |
